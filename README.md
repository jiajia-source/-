# jiajia-source 前端练习仓库
本仓库存放多个独立前端网页小项目，全部纯前端、无需后端，支持GitHub Pages直接在线打开。

## 项目清单
### 1. jia.html 待办清单页面
- 存储方式：浏览器localStorage本地持久化，刷新/关闭页面数据不会消失
- 默认内容：首次打开自动加载3条「想你」，第一条自带删除划线（已完成）
- 全部功能：输入文字新增待办、勾选切换完成状态、点击over删除条目、回车快捷添加
- 无第三方依赖：完全移除Supabase数据库，国内网络打开无加载报错
- 风格：粉色温柔渐变，电脑/手机自适应布局

### 2. index.html
通用空白前端模板，可用于新建网页练习

## 仓库说明
后续新增项目会在上方【项目清单】持续补充，每个html文件都是独立可直接运行的完整网页，下载后双击即可打开，也可部署GitHub Pages在线访问。
